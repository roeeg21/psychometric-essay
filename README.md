# Psychometric Writing Test Simulator

סימולטור מציאותי למבחן הכתיבה הפסיכומטרי בעברית

A realistic simulator for the Hebrew psychometric writing test.

## תכונות / Features

✅ **כתיבה מימין לשמאל** - תמיכה מלאה בכתיבה בעברית  
✅ **מספור שורות אוטומטי** - כל שורה ממוספרת באופן דינמי  
✅ **מגבלת שורות** - 25-50 שורות (מקסימום 55)  
✅ **טיימר מדויק** - ספירה לאחור עם התראה ב-5 דקות האחרונות  
✅ **כפתור סיום** - אפשרות לסיים את הכתיבה מוקדם  
✅ **דו"ח תוצאות** - סיכום מפורט של מספר מילים בכל שורה  
✅ **שמירה אוטומטית** - שמירת הטקסט כל 3 שניות ל-localStorage  
✅ **סרגל כלים** - העתקה, הדבקה וקו תחתון (כמו במבחן האמיתי)

## התקנה / Installation

### דרישות מקדימות / Prerequisites

- Node.js (גרסה 16 או גבוהה יותר)
- npm או yarn

### שלבי התקנה / Setup Steps

1. **שכפול/הורדת הפרויקט**
```bash
cd psychometric-test-simulator
```

2. **התקנת תלויות**
```bash
npm install
```

3. **הרצת שרת הפיתוח**
```bash
npm run dev
```

4. **פתיחת הדפדפן**
הפרויקט ייפתח אוטומטית בכתובת: `http://localhost:3000`

## בנייה לייצור / Build for Production

```bash
npm run build
```

הקבצים הסטטיים ייוצרו בתיקיית `dist/`

להרצת תצוגה מקדימה של גרסת הייצור:
```bash
npm run preview
```

## שימוש / Usage

### מסך בית

1. הזן את כמות הזמן הנוסף שקיבלת (אופציונלי)
2. לחץ על "לחץ Enter להתחלת הכתיבה" או הקש Enter

### מסך כתיבה

- כתוב את החיבור שלך מימין לשמאל
- השתמש בכלי ההעתקה וההדבקה בסרגל הכלים
- עקוב אחר מספר השורות והסטטוס בפינה השמאלית התחתונה
- לחץ "סיים כתיבה" כשסיימת או המתן עד שהזמן יסתיים

### מגבלות שורות

- **מינימום**: 25 שורות
- **מומלץ**: 25-50 שורות  
- **מקסימום מוחלט**: 55 שורות

המערכת תמנע ממך לחרוג מ-55 שורות.

### תוצאות

בסיום הכתיבה תוצג חלונית עם:
- סך כל השורות שנכתבו
- סך כל המילים
- פירוט מדויק של מספר המילים בכל שורה
